1. WAS 란?

Web Application Server 약자로 웹 애플리케이션을 실행하고 관리하는 서버를 나타냅니다.

DB조회나 다양한 로직 처리를 요구하는 동적인 컨텐츠를 제공하기 위해 만들어진 Application Serve이디.

대표적인 WAS로 Apache Tomcat이 있다.

서버 사이드에서 동작하여 클라이언트의 요청에 대응하고 결과를 반환하는 역할을 수행한다.

2. bit, byte 란?

bit는 가장작은 데이터 단위로 0,1의 이진(binary) 값으로 표현된다.

bit는 보통 true/false , yes/no , +/- , on/off로 사용되는 경우도 많다

byte는 가장 일반적인 8비트로 구성된 디지털 정보 단위다.

3. 톰켓 서버가 하는일?>

톰켓의 역할은 JSP/Servlet 엔진으로서 Application Server 역할을 한다.

자바 서블릿을 실행시키고 JSP코드가 포함되어 있는 웹페이지를 만들어준다.

4. 스프링프레임워크

동적인 웹 사이트를 개발하기 위한 여러 가지 서비스를 제공하고 있다.

대한민국 공공 기괸의 웹 서비스 개발 시 사용을 권장하고 있는 전자정부 표준프레임워크의 기반 기술로서 쓰이고 있다.
